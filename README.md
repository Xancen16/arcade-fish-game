# Ocean Dash 2D

Simple arcade game inspired by Subway Surfers, built with Python and Arcade library.

## Features
- 3-lane movement system.
- Persistent storage using **SQLite** (High Score and Coins).
- Dynamic difficulty: speed increases every 500 points.
- Character color change after collecting 10+ coins.

## Controls
- **UP Arrow**: Move up
- **DOWN Arrow**: Move down

## Installation
1. Install Python 3.10+
2. Install Arcade: `pip install arcade`
3. Run: `python main.py`
